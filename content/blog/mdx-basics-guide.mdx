MDX is Markdown + JSX. You can use regular Markdown syntax plus React components!

## Headings

Use `#` for headings:
- `#` H1 (use once per post)
- `##` H2
- `###` H3

## Text Formatting

You can use:
- **Bold text** with `**text**`
- *Italic text* with `*text*`
- `Inline code` with backticks

## Lists

### Unordered Lists
- Item 1
- Item 2
  - Nested item
  - Another nested item

### Ordered Lists
1. First item
2. Second item
3. Third item
4. Yes
